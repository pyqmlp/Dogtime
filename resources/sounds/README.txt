# 通知声音文件说明

在此目录下需要放置一个名为 "notification.wav" 的声音文件，用于倒计时结束时播放提示音。

## 建议的声音文件特性：
1. 文件格式: WAV 格式
2. 文件大小: 建议小于 300KB
3. 时长: 建议 1-3 秒
4. 声音特性: 简短、友好的提示音，与"狗"主题相关的声音（如轻柔的狗叫声）会更符合应用程序风格

## 获取声音文件的方法:
1. 使用免费的音效库网站如 Freesound.org
2. 使用系统自带的提示音
3. 录制自己的声音

请确保添加的声音文件没有版权问题。

如果没有添加此文件，应用程序将尝试使用系统默认提示音。
